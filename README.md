# adataviz

QueFaire? - Paris Events Explorer


Une application web interactive pour découvrir les derniers événements culturels à Paris, propulsée par l'Open Data de la Ville de Paris.



À propos du projet
Ce projet a été développé dans le cadre de ma formation à Ada Tech School. L'objectif est de manipuler l'API Open Data de Paris pour récupérer, traiter et afficher dynamiquement des événements culturels sur une interface web responsive.


🛠️ Technologies utilisées
HTML5 : Structure sémantique du contenu.
CSS3 : Mise en page (Grid/Flexbox) et design responsive.
JavaScript (ES6+) : Manipulation du DOM et logique asynchrone (async/await, fetch).
Vite : Outil de build et environnement de développement rapide.
Vitest : Framework de test unitaire.


✨ Fonctionnalités principales
Récupération dynamique : Connexion à l'API de la Ville de Paris pour afficher les 20 derniers événements.
Gestion des erreurs : Affichage sécurisé des données avec des valeurs par défaut pour les titres, images et descriptions.
Interface dynamique : Génération automatique de cartes (cards) via JavaScript.
Expérience utilisateur : Mise en page propre et moderne avec effets de survol (hover) sur les cartes.
